1) 숫자 함수
	ABS() -> 절대값(음수 -> 양수)
	CEIL() -> 올림
	FLOOR() -> 버림
	ROUND(숫자, 자리수) -> 반올림
	LOG()
	FOWER() -> 제곱
	SQRT() -> 제곱근
	SIGN() -> 음수 -> -1, 양수 1, 0->0
2) 문자 함수
	CONCAT(s1, s2) -> s1, s2 문자 결합
	LOWER(s) -> 대문자 -> 소문자
	UPPER(s) -> 소문자 -> 대문자
	LPAD(s, 갯수, 채울 문자)
	RPAD(s, 갯수, 채울 문자)
	
	REPLACE(문자, "찾을 문자(열)", "치환할 문자(열)")
	SUBSTATR(문자열, 시작위치, 자를 문자열 갯수)
	TRIM('제거할 문자' FROM '문자열');
	
	LENGTH
